# Лабораторная работа: Электронно-цифровая подпись

## Хэш-функция
- Разбиение на блоки по 16 бит
- Операция XOR с текущим хэшем
- Циклический сдвиг на 1 разряд влево

## Использование
1. Запустите un.bat
2. Программа автоматически:
   - Подпишет сообщение
   - Проверит подпись
   - Покажет, что изменение сообщения делает подпись недействительной

## Файлы
- digital_signature.py - исходный код
- un.bat - скрипт для запуска

## Результаты
| Сообщение | Хэш | Подпись | Результат |
|-----------|-----|---------|-----------|
| Оригинал (Петров) | 0x8A3F | 0x2095 | ✓ Подлинная |
| Изменено (Сидоров) | 0x1BC7 | 0x2095 (старая) | ✗ Неподлинная |
